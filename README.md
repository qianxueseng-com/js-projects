### 提交Pull request的办法
- fork这个repo，看见右上方的fork按钮了吗？点它。
- 去你自己fork完的repo，点"Clone in Desktop"，或者你用命令行也可
- 改代码，push到你自己fork完的repo
- 然后在你自己fork完的repo里面点Pull requests -> New pull request
